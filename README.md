# Next.js Climate Data Visualization

A modern climate data visualization platform built with Next.js, React, and Tailwind CSS.

## Features

- Interactive visualizations of global climate data
- Temperature anomalies, CO2 levels, renewable energy trends
- Responsive design for all devices
- Fast performance with Next.js optimization

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

### Build for Production

```bash
npm run build
npm start
```

## Project Structure

```
src/
  app/           # Next.js app directory
  components/    # Reusable React components
  hooks/         # Custom React hooks
  
public/
  data/          # CSV and JSON data files
  imgs/          # Image assets
```

## Data Sources

- NOAA Global Surface Temperature Dataset
- NOAA CO2 Trends
- Our World in Data - Renewable Energy
- NOAA Ozone Data
